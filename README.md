## Getting Started

First, install the dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Used

- Library: React
- Framework: Next.js
- CSS: Tailwind
- Component library: Shadcn/ui
